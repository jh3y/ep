progre(c)ss

CSS progress bars with minimal effort!
  • include the stylesheet
  • add the class progrecss to your element (any existing element)
  • Add a data attribute data-progrecss defining the completion percentage (1 to 100) and any optional data attributes and classes for your desired behaviour/position/appearance*
  • update progrecss value if necessary using scripts!
<div  class="progrecss" data-progrecss="77">your content!</div>
<div class="progrecss fixed blue" data-progrecss="75">fixed blue progre(c)ss</div>
<div class="progrecss mock fixed red" data-progrecss-mock="5">fixed red mocked progre(c)ss</div>
<div class="progrecss mock fixed staggered green" data-progrecss-mock="5">fixed staggered mock progre(c)ss</div>
<div class="progrecss timer fixed purple" data-progrecss-timer="10">timer progre(c)ss</div>

That's it!
