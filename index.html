progression

CSS progress bars with minimal effort!
  • include the stylesheet
  • add the class progression to your element (any existing element)
  • Add a data attribute data-progression defining the completion percentage (1 to 100) and any optional data attributes and classes for your desired behaviour/position/appearance*
  • update progre(c)ss value if necessary using simple vanilla js or whatever suits you best!
<div  class="progression" data-progression="77">your content!</div>
<div class="progression fixed blue" data-progression="75">fixed blue progre(c)ss</div>
<div class="progression mock fixed red" data-progression-mock="5">fixed red mocked progre(c)ss</div>
<div class="progression mock fixed staggered green" data-progression-mock="5">fixed staggered mock progre(c)ss</div>
<div class="progression timer fixed purple" data-progression-timer="10">timer progre(c)ss</div>

That's it!
